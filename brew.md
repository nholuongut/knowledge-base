## Install

```shell
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

## Package Management

```shell
brew install "$package"
```

```shell
brew reinstall "$package"
```

```shell
brew remove "$package"
```

## Info

```shell
brew info "$package"
```

List files for package:

```shell
brew ls --verbose "$package"
```
